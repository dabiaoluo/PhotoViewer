# PhotoViewer
Image viewer written in Python 2.7 + PyQt 4. It supports multiple image formats.

####Features

* Next/Previous picture
* Zoom in/out
* Fullscreen
* Rotate
* Set to wallpaper
* Support GIF animation
* Auto fit window size
* Frameless window
* Multiple theme
