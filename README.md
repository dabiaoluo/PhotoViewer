# PhotoViewer
Image viewer code by Python 2.7 + PyQt 4. Multiple image formats support.

## Features

* Next/Previous picture
* Zoom in/out
* Fullscreen
* Rotate
* Set to wallpaper
* Support GIF animation
* Auto fit window size
* Frameless window
* Multiple theme
